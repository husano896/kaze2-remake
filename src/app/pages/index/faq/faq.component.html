FAQ

[IOS] 為何我的遊戲在接耳機時才有聲音？ / Why my game only have sounds when headphones are connected? iPhone / iPad的靜音模式會使遊戲無聲音，請關閉後才能從喇叭發出聲音。
iPhone / iPad has silent mode, please turn off to play sounds from the speaker.

No, the new version use a totally different framework and server. And I don't migreate saves from old server so the server is fresh new. Migration is still to be discussed, maybe later...?
